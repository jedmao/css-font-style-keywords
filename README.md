# css-font-styles

List of CSS [font styles](https://developer.mozilla.org/en-US/docs/Web/CSS/font-style):
- normal
- italic
- oblique

## Installation

```
npm install css-font-styles [--save[-dev]]
```

## Usage

```js
var cssFontStyles = require('css-font-styles');
cssFontStyles; // ['normal', 'italic', 'oblique']
```
